
# GYMADMIN

A responsive web application that all shows all the users from the Random User API.


## Languages and Tools:

Development:
- React.js 
- HTML
- CSS

  

## Run Locally

Clone the project

```bash
  git clone https://github.com/pilimartinez/user-app.git
```

Go to the project directory

```bash
  cd gymadmin
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
